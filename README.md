# TecnoLógica

Site desenvolvido durante a graduação em Engenharia de Software com o objetivo de praticar a criação de páginas web utilizando HTML e CSS.

## Tecnologias utilizadas

* HTML5
* CSS3

## Funcionalidades

* Página inicial com notícias sobre tecnologia
* Página dedicada ao universo dos jogos
* Página com conteúdo sobre smartphones
* Menu de navegação entre páginas
* Utilização de imagens e conteúdo organizado por categorias
* Layout personalizado com CSS externo

## Objetivo

Aplicar conceitos fundamentais de desenvolvimento web, incluindo estruturação de páginas, navegação entre seções, organização de conteúdo, estilização com CSS e criação de interfaces para o usuário.

## Estrutura do projeto

* Home
* Jogos
* Smartphones

## Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos de HTML semântico, links de navegação, estilização com CSS, organização de arquivos e construção de layouts para páginas web.
